Javascript Definitions:


Define function

Functions are a set of organised instructions that correspond to a certain task or specific functionality a user wants to implement in their program to get a single desired outcome.


What is a function?

Functions allow you to reuse code. You can use the same function with different arguments to produce different results. Functions is defined with the function keyword followed by a name, followed by parentheses.

Define variable

A variable is a container that stores data. It allows programmers to reuse, update or to keep track of your values. Variables are essential for storing and manipulating data in computer programs.


Define data types

Data types classify the kind of value a variable holds and define the operations that can be performed on it. There are 7 data types and they are:

  • String- which represents a series of characters
  • Real- which stores decimal numbers
  • Boolean -which represents true or false, yes or no
  • Array- which holds a collection of values
  • Function- which performs a certain task
  • object- that can contain built-in objects and user defined objects
  • undefined- which represents an uninitialized variable
  • null- indicates the absence of any value
